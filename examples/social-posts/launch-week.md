# Launch Week Posts

## Day 1: The Big Launch

Most AI agents can write a caption. Very few can run a full marketing workflow.

That is why I built `marketing-skills`: a full Agent Skills library for strategy, SEO, content, email, CRO, sales, PR, local SEO, ecommerce, prospecting, integrations, and CLI-powered workflows.

Use it with Claude Code, OpenAI Codex, Cursor, Windsurf, and any Agent Skills-compatible agent.

Install:
```bash
npx skills add scayver/marketing-skills
```

Star the repo: https://github.com/scayver/marketing-skills

## Day 2: Marketing OS

Your AI agent should not guess what marketing task comes next.

The `marketing-os` skill routes work across strategy, demand, conversion, pipeline, retention, measurement, and assets.

It turns a generic coding agent into a marketing operating system that knows which skill, integration guide, or CLI helper to use for the goal in front of it.

Install:
```bash
npx skills add scayver/marketing-skills --skill marketing-os
```

## Day 3: Developer Intent

Developer intent is hiding in public.

Stars, forks, watchers, contributors, hiring signals, and repo activity can all show who cares about a problem before they fill out a form.

The library includes prospecting skills, GitHub intent workflows, and a `github-prospects.js` CLI so agents can turn developer activity into structured outbound research.

Install the repo and try the GitHub prospecting workflow.

## Day 4: SEO Growth Engine

SEO is not just keywords anymore.

You need briefs, content strategy, SERP analysis, schema, AI search visibility, repurposing, and measurement.

`marketing-skills` gives your agent SEO skills plus standalone integration guides for tools like Ahrefs, Semrush, Search Console, DataForSEO, and GA4.

Install once. Give your agent a real growth workflow.
