# Scheduling

## 1. calendly

**Type:** cli
**Category:** Scheduling
**Platform:** LinkedIn / X

**Hook**
`calendly.js` turns agent advice into executable marketing ops.

**Post**
This zero-dependency Node.js CLI gives your agent a practical helper for scheduling command-line helper for Calendly. It outputs JSON, supports scriptable workflows, and fits Claude Code, OpenAI Codex, Cursor, and Windsurf projects where marketing work needs real platform actions instead of another checklist.

**CTA**
Try it: `node tools/clis/calendly.js --dry-run`. Install Marketing Skills OS: npx skills add scayver/marketing-skills. Repo: https://github.com/scayver/marketing-skills.

**Install**
`npx skills add scayver/marketing-skills`
