# Referral

## 1. mention-me

**Type:** cli
**Category:** Referral
**Platform:** LinkedIn / X

**Hook**
`mention-me.js` turns agent advice into executable marketing ops.

**Post**
This zero-dependency Node.js CLI gives your agent a practical helper for referral command-line helper for Mention Me. It outputs JSON, supports scriptable workflows, and fits Claude Code, OpenAI Codex, Cursor, and Windsurf projects where marketing work needs real platform actions instead of another checklist.

**CTA**
Try it: `node tools/clis/mention-me.js --dry-run`. Install Marketing Skills OS: npx skills add scayver/marketing-skills. Repo: https://github.com/scayver/marketing-skills.

**Install**
`npx skills add scayver/marketing-skills`
