# Reviews

## 1. g2

**Type:** cli
**Category:** Reviews
**Platform:** LinkedIn / X

**Hook**
`g2.js` turns agent advice into executable marketing ops.

**Post**
This zero-dependency Node.js CLI gives your agent a practical helper for reviews command-line helper for G2. It outputs JSON, supports scriptable workflows, and fits Claude Code, OpenAI Codex, Cursor, and Windsurf projects where marketing work needs real platform actions instead of another checklist.

**CTA**
Try it: `node tools/clis/g2.js --dry-run`. Install Marketing Skills OS: npx skills add scayver/marketing-skills. Repo: https://github.com/scayver/marketing-skills.

**Install**
`npx skills add scayver/marketing-skills`
