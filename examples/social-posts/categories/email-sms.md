# Email/SMS

## 1. brevo

**Type:** cli
**Category:** Email/SMS
**Platform:** LinkedIn / X

**Hook**
`brevo.js` turns agent advice into executable marketing ops.

**Post**
This zero-dependency Node.js CLI gives your agent a practical helper for email/sms command-line helper for Brevo. It outputs JSON, supports scriptable workflows, and fits Claude Code, OpenAI Codex, Cursor, and Windsurf projects where marketing work needs real platform actions instead of another checklist.

**CTA**
Try it: `node tools/clis/brevo.js --dry-run`. Install Marketing Skills OS: npx skills add scayver/marketing-skills. Repo: https://github.com/scayver/marketing-skills.

**Install**
`npx skills add scayver/marketing-skills`
