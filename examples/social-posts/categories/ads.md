# Ads

## 1. google-ads

**Type:** cli
**Category:** Ads
**Platform:** LinkedIn / X

**Hook**
`google-ads.js` turns agent advice into executable marketing ops.

**Post**
This zero-dependency Node.js CLI gives your agent a practical helper for ads command-line helper for Google Ads. It outputs JSON, supports scriptable workflows, and fits Claude Code, OpenAI Codex, Cursor, and Windsurf projects where marketing work needs real platform actions instead of another checklist.

**CTA**
Try it: `node tools/clis/google-ads.js --dry-run`. Install Marketing Skills OS: npx skills add scayver/marketing-skills. Repo: https://github.com/scayver/marketing-skills.

**Install**
`npx skills add scayver/marketing-skills`
