# Social Posting Kit

Local copy-ready launch assets for the marketing-skills repo.

Install command to include in posts:

```bash
npx skills add scayver/marketing-skills
```

Repo link: https://github.com/scayver/marketing-skills

## What is here

| File | Use |
|------|-----|
| `30-day-posting-calendar.csv` | Daily posting plan with selected asset, hook, CTA, and install command. |
| `launch-week.md` | Seven polished posts for the first week. |
| `category-stacks.md` | Mixed category posts that bundle skills, integrations, and CLI tools. |
| `skills-posts.md` | One post for each skill. |
| `integration-posts.md` | One post for each integration guide. |
| `cli-tool-posts.md` | One post for each CLI tool. |
| `categories/` | Smaller category-specific batches for faster posting. |

## Posting rhythm

- Monday: category stack or Marketing OS post.
- Tuesday: individual skill post.
- Wednesday: integration guide post.
- Thursday: CLI tool post.
- Friday: workflow or demo post.
- Saturday: build-in-public/founder post.
- Sunday: recap and install CTA.
