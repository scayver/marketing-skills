# Code of Conduct

## Standard

This project welcomes contributors who want to improve agent skills, marketing workflows, integration references, and CLI helpers.

Be direct, respectful, and useful. Focus criticism on the artifact, not the person. Keep issues and pull requests specific enough that another contributor can act on them.

## Unacceptable Behavior

Unacceptable behavior includes harassment, personal attacks, doxxing, spam, discriminatory language, and submitting malicious code or documentation intended to expose credentials or violate platform terms.

## Enforcement

The maintainer may edit, hide, or remove comments and issues that violate this standard. Repeated or severe violations may result in blocked participation.
