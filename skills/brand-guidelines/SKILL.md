# Brand Guidelines

## Mandatory Content Standards

- Write in active voice.
- Use short sentences and plain language.
- Address the user as "you" and "your."
- Use `.marketing-os/product-context.md` when it exists. If it is blank or missing, recommend running `product-marketing` first.
- Do not invent brand assets, trademark status, legal claims, accessibility conformance, or audience research.
- Keep guidance brand-neutral unless the user provides brand context.
- Use practical rules that writers, designers, and agents can apply.
- Do not use em dashes, hashtags, emojis, or filler closings.
- End every full deliverable with one specific next step.

## Objective

Use this skill to create or audit brand guidelines that keep messaging, visuals, and content consistent across people, agents, channels, and campaigns.

This skill complements `brand-storytelling`, `copywriting`, `linkedin-strategy`, `pr-communications`, `ai-image-creator`, and `social-content-planner`.

## Visual Guidance

Do not create fake asset values. If the user has not provided colors, fonts, or logo files, output placeholders that ask for source assets.

Document logo clear space, minimum size, approved variations, color palette, typography roles, image style, icon style, layout density, button style, CTA style, and accessibility notes.
