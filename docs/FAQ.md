# FAQ

## What is this repo?

Marketing Skills OS is an open-source Marketing OS for AI agents. It gives AI coding agents skills, workflows, integration guides, CLI helpers, templates, dashboards, scorecards, and review loops for running marketing work from strategy through execution.

Compatibility note: the install path remains `scayver/marketing-skills`.

## Who is it for?

Founders, marketers, agencies, creators, ecommerce teams, SaaS builders, and anyone using AI agents to ship marketing work faster.

## Which agents does it support?

It is designed for Claude Code, OpenAI Codex, Cursor, Windsurf, and any agent that supports the Agent Skills spec.

## How do I install it?

```bash
npx skills add scayver/marketing-skills
```

## Do I need every skill?

No. You can install all skills or specific ones:

```bash
npx skills add scayver/marketing-skills --skill copywriting landing-page-cro email-marketing
```

## What should I run first?

Start with `product-marketing`. It creates the context other skills use for product, audience, positioning, and value proposition.

## What are tool references?

Tool references are markdown guides in `tools/integrations/`. They help agents work with APIs, CLIs, SDKs, WordPress REST endpoints, webhooks, and platform-specific fields.
