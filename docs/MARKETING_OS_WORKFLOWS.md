# Marketing Skills OS Workflow Index

This index turns common business goals into skill chains, tool references, CLI helpers, outputs, and metrics.

Use it when you know the desired business outcome but do not know which skill to activate first.

## SaaS Launch

**Goal:** Launch a SaaS product, feature, AI tool, open-source project, or agent product.

**Primary skill:** `marketing-os`.

| Area | Items |
|------|-------|
| Skills | `product-marketing`, `business-strategy`, `launch`, `landing-page-cro`, `email-marketing`, `linkedin-strategy`, `pr-communications`, `analytics` |
| Integration guides | `github`, `ga4`, `hubspot`, `mailchimp`, `customer-io`, `activecampaign` |
| CLI helpers | `github-prospects`, `ga4`, `mailchimp` |
| Deliverables | positioning context, launch plan, launch page, email sequence, social posts, press announcement, tracking plan |
| Metrics | visitors, waitlist signups, trial starts, activation, demos, revenue |

**Install the workflow skills:**

```bash
npx skills add scayver/marketing-skills --skill product-marketing business-strategy launch landing-page-cro email-marketing linkedin-strategy pr-communications analytics
```
