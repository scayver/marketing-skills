# Showcase

Use these workflows to demonstrate what Marketing Skills OS can do.

## Workflow 1: Launch a SaaS Product

Use:

- `product-marketing`.
- `launch`.
- `email-marketing`.
- `linkedin-strategy`.
- `pr-communications`.

Prompt:

```text
Create a complete launch system for my SaaS. Start with product positioning, then create a launch plan, 5-email sequence, LinkedIn announcement posts, and a short press release.
```

Expected output:

- Positioning context.
- Launch timeline.
- Email sequence.
- LinkedIn posts.
- Press release.
- CTA map.

## Workflow 2: Fix a Low-Converting Landing Page

Use:

- `landing-page-cro`.
- `copywriting`.
- `ab-testing`.
- `analytics`.

Prompt:

```text
Audit this landing page for conversion problems. Rewrite the hero, benefits, CTA, objections, trust proof, and FAQ. Then create 3 A/B test ideas.
```

Expected output:

- Page friction analysis.
- Rewritten copy.
- FAQ schema suggestions.
- A/B test backlog.
- Measurement plan.
