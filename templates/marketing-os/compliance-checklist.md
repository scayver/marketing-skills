# Compliance Checklist

This is a working reference, not legal advice. Confirm requirements with qualified counsel when risk is high.

## Email and outbound

| Area | Question | Status | Notes |
|------|----------|--------|-------|
| CAN-SPAM | Does the email identify the sender and include a working opt-out? | Unknown | |
| GDPR | Do we have a lawful basis for processing EU personal data? | Unknown | |
| CASL | Do we have consent or a valid exemption for Canadian recipients? | Unknown | |
| Platform ToS | Does this workflow follow platform terms for scraping, automation, and outreach? | Unknown | |
| Data minimization | Are we collecting only the fields needed for the campaign? | Unknown | |

## Website and analytics

| Area | Question | Status | Notes |
|------|----------|--------|-------|
| Cookies | Are tracking scripts disclosed where required? | Unknown | |
| Consent | Is consent captured before non-essential tracking where required? | Unknown | |
| Privacy policy | Does the policy match actual collection and use? | Unknown | |

## Agent prompt

```text
Use the compliance reference and the active workflow. Identify compliance risks, missing disclosures, consent gaps, and safer alternatives before execution.
```
