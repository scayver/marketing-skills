# n8n

n8n is an open-source, self-hostable workflow automation tool with a node-based visual editor and support for custom code. Available as an Action in the Bit Integrations WordPress plugin.

**Role:** Action
**Free Tier:** Yes
**Category:** Automation and Integration Platforms
**Icon:** `https://bit-integrations.com/wp-content/uploads/2026/02/n8n.svg`

## Capabilities in Bit Integrations

| Feature | Available | Notes |
|---------|-----------|-------|
| As Trigger | — | — |
| As Action | ✓ | Send data to an n8n Webhook node to start a workflow |
| Free Tier | ✓ | Free with Bit Integrations free plan |
| Field Mapping | ✓ | Map any form field to the outgoing webhook payload |

## Action Events

- Send data to an n8n Webhook node to trigger a self-hosted or cloud n8n workflow

## Authentication

- **Type**: Webhook URL
- **Required**: n8n Webhook URL generated by the Webhook node in an n8n workflow

## Common Workflow Recipes

### Recipe 1: Form Data to Custom Processing Pipeline
**Trigger:** WordPress form submission
**Action:** n8n workflow receives data, runs JavaScript transformations, and routes to multiple destinations
**Use case:** Apply custom business logic to form data before sending to CRM, spreadsheet, or notification services

### Recipe 2: WooCommerce Order to Custom Fulfillment API
**Trigger:** WooCommerce order completed
**Action:** n8n workflow parses order data and calls a custom REST API for fulfillment
**Use case:** Connect WooCommerce to proprietary fulfillment or ERP systems that lack standard connectors

### Recipe 3: User Registration to Internal Database
**Trigger:** WordPress user registration
**Action:** n8n workflow writes user record to an internal PostgreSQL or MySQL database
**Use case:** Keep an internal user database synchronized with WordPress registrations

## Setup Steps

1. Install Bit Integrations on your WordPress site.
2. Go to Bit Integrations > Create Integration.
3. Choose your trigger.
4. Select n8n as the Action.
5. In n8n, create a workflow and add a "Webhook" node. Activate the workflow and copy the webhook URL.
6. Paste the webhook URL into Bit Integrations.
7. Map fields.
8. Save and test.

## When to Use

- When you want self-hosted automation with full data privacy and no per-task pricing
- When you need custom JavaScript logic, database queries, or API calls as part of the automation
- When connecting WordPress to internal or enterprise systems not covered by SaaS automation tools
