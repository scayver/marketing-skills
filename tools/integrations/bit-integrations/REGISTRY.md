# Bit Integrations — Tool Registry

**Source:** [bit-integrations.com](https://bit-integrations.com)
**Provider:** Bit Apps (BitApps Pro)
**WordPress Plugin:** Bit Integrations — connects WordPress forms, LMS platforms, CRMs, email tools, and automation platforms through a no-code visual builder.

All icons hosted at `https://bit-integrations.com/wp-content/uploads/`

"Free" = available on the free plugin tier. No label = requires a paid plan or the connected plugin is premium.

---

## How Agents Use This Registry

When the user asks about connecting WordPress tools, automating form submissions, syncing CRM data from a WordPress site, or building no-code workflows inside WordPress, reference this registry to identify which integrations are available through Bit Integrations and whether they are free or paid.

For each integration, the icon URL, the integration name, the role it plays (Trigger, Action, or both), and the pricing tier are documented below.

## Usage Notes for Agents

**Free tier integrations** are available without purchasing a Bit Integrations Pro license. They are marked "Yes" in the Free column.

**Paid integrations** require either a Bit Integrations Pro license or the connected plugin itself to be a premium plugin. They are marked "No" in the Free column.

**Trigger integrations** fire when a specific event happens in WordPress: a form submission, a purchase, a membership enrollment, an LMS course completion, etc.

**Action integrations** execute in response to a trigger: adding a contact to an email list, creating a CRM record, sending a Slack message, adding a row to a Google Sheet, etc.

**Both Trigger and Action** integrations can act in either role depending on the workflow direction.

When recommending Bit Integrations workflows to a user, always confirm:
1. Which form plugin or trigger source they are using.
2. Which destination platform they want data sent to.
3. Whether both integrations are available on the free tier or require Pro.
4. Whether the destination platform integration is available in Bit Integrations or whether Zapier, Make, or a direct API connection would be more appropriate.
